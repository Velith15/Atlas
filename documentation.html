System Overview

Welcome to the Atlas documentation. Build, deploy, and manage intelligent workflows at scale.

What is Atlas?

Atlas is a unified platform for architecting automation—combining a visual workflow builder, a powerful execution engine, and native AI.

Atlas is currently in public beta. Features and APIs will evolve based on feedback.

How it works

Atlas workflows consist of three primary components:

  1. Triggers: Events that start the execution.
  2. Nodes: Logic, data transformations, or AI operations.
  3. Actions: Final outputs or external API calls.
